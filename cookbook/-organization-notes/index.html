-organization notes

  • within the recipes folder, there are individual markdown notes with each recipe I have made or want to make
    • each recipe has a Source at the bottom, so as to be able to find the original recipe
  • these recipes get saved to a separate markdown file to allow for easy modifications after cooking
    • generally these modifications will be made by adding an arrow -> after whatever step or ingredient I changed, with the change in bold
  • some recipes will also have photos at the bottom showing finished products
  • I use ChatGPT to convert the web recipe to a markdown recipe, using [[-chatgpt recipe prompt|this prompt]]

sorting

  • each recipe has a number of tags in their file properties, split into different categories
  • tags for what kind of meal it is
    • breakfast, lunch, dinner, appetizer
  • tags for main / secondary ingredient
    • beef, chicken, rice
